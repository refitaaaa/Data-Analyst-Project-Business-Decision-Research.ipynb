# Data-Analyst-Project-Business-Decision-Research.ipynb
Analisis data transaksi ritel untuk memahami perilaku pelanggan dan memprediksi risiko churn. Menggunakan Python, visualisasi, dan Logistic Regression, model memprediksi pelanggan berisiko dengan akurasi 76%, mendukung strategi retensi dan pengambilan keputusan berbasis data.
